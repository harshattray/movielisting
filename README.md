## Installation Instructions

=========================

## Initial Project Setup

Project Initialisation involves standard procedures listed below

Clone the repo

      git clone https://github.com/harshattray/movielisting.git

Install npm modules

      npm install --save

Install dev dependencies

      npm install --save-dev

## Run the project

      npx run android - To run the Project on android  device

      npx run  iOS - To run the tests on iOS device
